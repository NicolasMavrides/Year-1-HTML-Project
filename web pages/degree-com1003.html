    My Degree/COM1003

    COM1003 (Java Programming) is a module which covers material on programming using the Java language. The module has 1 lecture per week which covers the theory, and 2 lab sessions, during which support is provided on solving various exercise sheets.

    The module grade is assessed by means of assignments. Each assignment instructs an individual to complete a program in Java and the individual is assessed based on how much the final program meets the required specifications.
