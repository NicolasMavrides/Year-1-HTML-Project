    My Degree/COM1008

    COM1008 (Web and Internet Technology) is a module which covers material mainly on website design and implementation. The module has 2 lectures per week on which theory is covered. Theory includes markup languages (HTML and CSS), use of JavaScript, Networks and Protocols and Security.

    The module also has a lab session every Friday during which either exercise sheets are covered or quizzes are held. Lab sheets include working on examples with the use of the markup languages and JavaScript in web design (pictured), and support is provided during lab sessions.

    The COM1008 module is assessed by means of quizzes along with a website assignment, which is this website itself.
