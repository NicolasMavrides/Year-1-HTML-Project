    Accessibility statement

    I have made every attempt possible to make this website as accessible as possible to the widest audience range by providing an alternative, mobile version for users on smaller devices, and by conforming to the guidelines of W3C technologies so that the website code is compatible with most major browsers.

    The website has been tested and trialled with Google Chrome, Firefox, and Internet Explorer browsers and has found to be working on the current versions of all three. By conforming to the W3C standards, this should ensure that the website will remain compatible with future versions of the browsers it has been tested on.

    Despite my attempts, it is still possible that an area of accessibility may be left out. If you happen to experience difficulty accessing the website, please don't hesitate to contact me.
