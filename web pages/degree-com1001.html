    My Degree/COM1001

    COM1001 (Software Engineering) is a module which covers aspects and methods of Software Development processes. The aim of the module is to introduce the theory of developing a system and ensuring a system meets all its goals.

    The module has 2 scheduled lectures per week which cover the theory for software development. The module also has 1 lab session every Wednesday during which exercise sheets on programming with the Ruby language are covered.

    In addition, the module also covers a group project which spans over both semesters, during which a group is required to work together to implement a software system for a "client", via methods taught in lectures or otherwise and with programming knowledge gained from the lab sessions
